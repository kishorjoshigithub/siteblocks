## SiteBlocks

SiteBlocks is an AI-powered web website builder that enables users to design and iterate on websites using natural language prompts. Simply describe the website you want, and SiteBlocks generates a responsive layout instantly.

Users can refine designs through additional prompts, regenerate specific sections, or manually edit components for complete flexibility. SiteBlocks bridges the gap between AI-driven creation and hands-on customization, making website building faster, smarter, and more accessible.

### Features

- AI-powered website generation from text prompts
- Prompt-based revisions and section-level regeneration
- Manual editing for full design control
- Responsive, modern layouts
- Fast iteration without writing code
- Credit-based usage model for creation and revisions
